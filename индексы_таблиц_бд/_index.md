 **Индекс** ‑ объект базы данных, создаваемый с целью повышения производительности поиска данных.
Индексы:
- простой (одно поле)
- составной (несколько полей)
**Кластерный индекс**
При наличии кластерного индекса строки таблицы физически упорядочены по значению ключа этого индекса. Кластерный индекс может быть только один для каждой таблицы.
**Некластерный индекс**
Некластерный индекс, созданный для таблицы, содержит только указатели на записи таблицы. Каждая таблица может иметь несколько различных некластерных индексов, каждый из которых определяет свой собственный порядок следования записей.
